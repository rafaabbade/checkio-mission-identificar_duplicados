Como encontrar a posição de itens em uma lista sem criar uma variável adicional?

Você pode usar a função enumerate() para obter tanto o índice quanto o valor de cada item em uma lista durante a iteração. Isso elimina a necessidade de criar uma variável adicional para contar a posição dos itens. Se quiser, teste o código abaixo no online-python.com 

numeros = [10, 20, 30, 40]
for indice, valor in enumerate(numeros):
    print(f"Índice: {indice}, Valor: {valor}")
