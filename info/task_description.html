Enunciado

Paulo, um engenheiro de software, precisa encontrar todas as ocorrências de um ID específico de usuário em uma lista para resolver um problema de duplicação de dados. Escreva uma função que receba uma lista de inteiros e um valor inteiro. A função deve retornar uma lista de índices onde o valor aparece na lista.

Entrada: Uma lista de inteiros e um valor inteiro.

Saída: Uma lista com o índice dos elementos duplicados.
